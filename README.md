# Create Site in cloudflare

<p>Requirements to run application`</p>
<ul>
    <li>Node >= 16 or docker with docker-compose</li>
</ul>


<p>Launch api(with docker) run this command`</p>

```
docker-compose up
```

<p>Launch api(with node) run this command`</p>

```
npm install
node app.js
```
